官方网站：https://styleci.io/

Style CI 是专门针对于 PHP 的持续集成工具，具有 `代码格式化` 等功能。

<!--more-->

在项目根目录增加 `.styleci.yml` 文件，并在 StyleCI 官网打开项目构建开关即可开始使用。

示例：https://github.com/khs1994-php/tencent-ai
