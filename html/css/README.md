# 元素分类

## 块状元素

`display:block`

## 内联元素（行内元素）

`display:inline`

和其他元素都在一行上；元素的高度、宽度及顶部和底部边距不可设置；元素的宽度就是它包含的文字或图片的宽度，不可改变。

## 内联块状元素

`display:inline-block`

和其他元素都在一行上；元素的高度、宽度、行高以及顶和底边距都可设置。
