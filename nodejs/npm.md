npm 是 Node.js 的包管理工具。

<!--more-->

# 换源

```bash
$ echo "registry=https://registry.npm.taobao.org" >> ~/.npmrc
```

## npm 升级

```bash
$ npm i -g npm
```

# 常用工具

npm-check

```bash
$ npm install -g npm-check
```
